# AuditUX
Web Tool depicted to analyze UX of webp pages through its link. Built with NextJs and powered by LLM. It is based in the accomplishment of the Nielsen Heuristics.


## Getting Started

To start locally:

```bash
pnpm install
pnpm dev
```

Open [http://localhost:8080](http://localhost:8080) with your browser to see the result.


## Deployment
The deployment is automated to a free plan of Vercel Platform.
