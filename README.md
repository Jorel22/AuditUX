# AuditUX
Herramienta web diseñada para analizar la experiencia de usuario (UX) de páginas web a través de sus enlaces. Desarrollada con Next.js y basada en LLM, se fundamenta en la metodología heurística de Nielsen.


## Para empezar

Para iniciar localmente:

```bash
pnpm install
pnpm dev
```

Abrir [http://localhost:8080](http://localhost:8080) con tu navegador para ver el resultado.


## Despliegue
El despliegue está automatizado a Vercel.
